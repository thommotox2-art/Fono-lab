# 🌊 FonoClínica - Landing Page Premium

Uma Landing Page moderna e de altíssimo padrão (Design Vibe 2026) criada para clínicas de **Fonoaudiologia**, com foco em conversão de pacientes, bem-estar e humanização.

## ✨ Características Principais

* **Design Responsivo e Premium:** Interface fluída com efeitos de *glassmorphism*, sombras complexas e gradientes orgânicos, transmitindo acolhimento e excelência clínica.
* **Tecnologias Utilizadas:** HTML5 semântico e **Tailwind CSS** (via CDN) com configurações customizadas de tema e animações (Exemplo: bg animados).
* **Seções Otimizadas:**
  * **Hero (Início):** Chamada para ação (CTA) marcante com foco na confiança e liberdade da comunicação.
  * **Especialidades:** Cards interativos utilizando a biblioteca de ícones *Phosphor Icons* para demonstrar serviços (Linguagem Infantil, Voz Profissional e Reabilitação Auditiva).
  * **A Profissional:** Layout atrativo com foto realista e biografia que reflete empatia e formação científica.
  * **Depoimentos:** Exposição limpa de casos de sucesso para construção de prova social (Social Proof).
* **Imagens AI Customizadas:** Fotografias de um consultório moderno e um retrato de rosto profissional gerados especificamente sob medida com Inteligência Artificial e alocados na pasta oculta `/assets`.

## 🚀 Como Executar o Projeto

Como o site foi simplificado estruturalmente com HTML padrão e injetando o Tailwind diretamente por script tag (CDN), não é exigida nenhuma configuração com `Node.js` ou compiladores:

1. Clone o repositório em sua máquina:
   ```bash
   git clone https://github.com/thommotox2-art/Fono-lab.git
   ```
2. Acesse a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador web, ou utilize ferramentas como a extensão **Live Server** (do VS Code) para ter _hot-reload_ enquanto faz modificações.

## 🎨 Cores e Estilo Visual

A paleta visual engloba as características da saúde auditiva e fonatória (Tons de Azul suave e Branco), focando na sensação reconfortante (variação do tom primário `$primary: #0f8df0` até tons _slate_ calmos para texto textual).
A tipografia de marca selecionada, **Outfit**, assegura um equilíbrio entre o familiar (bordas arredondadas) e a precisão da era digital.
